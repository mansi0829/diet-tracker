# Diet tracker

How it will work,

- ```
  git clone (https://github.com/mansi0829/diet-tracker.git)
  ```
- ```
  cd diet-tracker
  ```
- ```
  python3 -m venv venv 
  ```
- ```
  source venv/bin/activate 
  ```
- ```
  pip install -r requirements.txt
  ```
- ```
  python manage.py runserver
  ```

- Then Open `http://127.0.0.1:8000/`
